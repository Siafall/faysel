Présentation de l'UTEC

L'UTEC est un centre de formation professionalisante pour les apprentis proposant plusieurs diplômes comme le BTS SIO où on peut choisir entre deux options :

SISR (Solutions d’Infrastructures Système et Réseau) qui permet de se professionaliser dans la maintenance d'infrastructures réseau et système comme son nom l'indique

SLAM (Solutions Logicielles et Applications Métier) qui permet de se professionaliser dans le développement logiciel et applicatif pour les entreprises

Présentation d'Atempo

Atempo est une entreprise fondée en 1992 sous le nom de Quadratec, qui voit son nom changé pour Atempo en 2001. Cette entreprise est spécialisée dans la sauvegarde, l'archivage et le déplacement de données. Je suis chargé de transitionner les anciens fichiers (Makefile) vers un nouveau format (CMake).
